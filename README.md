# map-navigator
Интерактивная карта России

1. Отображение городов Росиии и информации о них на карте.
2. Авторизация с хешированием паролей (bcrypt).
3. Двухфакторная аутенфикация (код на email).
4. Расчёт расстояния по формуле Гаверсинуса.
5. Расчёт времени пути, количества и стоимость топлива.
6. Сохранение избранных маршрутов.

Backend  Python + FastAPI 
Database PostgreSQL
Frontend HTML/CSS/JavaScript + Leaflet 
Безопасность bcrypt, 2FA через email 
Функциональный модуль C++ (расчёт расстояния по формуле Гаверсинуса)
